APLIKASI DATA KARYAWAN

1. Jalankan XAMPP.
2. Start Apache dan MySQL.
3. Salin folder aplikasi_data_karyawan ke:
   C:\xampp\htdocs\
4. Buka http://localhost/phpmyadmin
5. Klik Import, pilih database.sql, lalu klik Go/Kirim.
6. Buka:
   http://localhost/aplikasi_data_karyawan
7. Login:
   Username: admin
   Password: admin123

Fitur:
- Login dan logout
- Dashboard
- CRUD tabel karyawan
- CRUD tabel users
- Pencarian karyawan
- Bootstrap dan CSS
